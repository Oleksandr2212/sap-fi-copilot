# Open Data Sources

Ця папка містить зовнішню (публічну) базу знань для SAP FI Copilot.

Використані відкриті джерела:
- https://www.sapdocs.info/application-modules/fi-co-overview/fi-transaction-codes/
- https://aboutsapfi.blogspot.com/2013/01/sap-fi-financial-accounting-fi_4709.html

Файл `transactions_open.json` містить додаткові FI транзакції з українськими описами, нормалізованою структурою та полем `source`.
